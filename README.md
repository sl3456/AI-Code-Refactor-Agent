# AI Code Refactoring Agent (Demo)

This project implements an automated AI Agent capable of scanning technical debt and suggesting refactors.

## Features
- **Scanner Agent**: Identifies code smells in local Python files.
- **Refactor Agent**: Generates optimized code versions using LLMs.
- **Verification Logic**: Simulates a self-reflection loop to validate changes.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Set your API Key: `export API_KEY='your_key_here'`
3. Run the agent: `python main.py`
